# Odoo-Install
Script para instalar odoo

Para instalar odoo se requiere de:
-Ubuntu 22.04 LTS
-Python 3.7 o superior 

```
python3 --version
```


-pip (Instalador de librerias)
```
pip3 --version
```



## 1.- Clonar repositorio en servidor
 ```
git clone https://github.com/AlexRiosXI/Odoo-Install
```

```

```

